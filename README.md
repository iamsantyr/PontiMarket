# PontiMarket

**PontiMarket** is a mobile application built with Swift and Xcode, designed to streamline buying and selling within a university community. It is a marketplace tailored for student entrepreneurs who want to offer their products in an organized, secure, and accessible environment.

---

## Features

- Secure authentication for university members  
- Product browsing by categories  
- Product publishing by student sellers  
- Instant contact between buyers and sellers  
- Multi-language support (English & Spanish)
